To run this application, execute the following commands:

    ```

    ```

  2. Start the front-end dev server:

    ```
    $ npm run dev
    ```

    ```
    $ NODE_ENV=production grommet pack
    ```

  4. Test and run linters:

    ```
    $ npm test
    ```
